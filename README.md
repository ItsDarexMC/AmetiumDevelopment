## Ametium Development

Ametium Development es el proyecto principal enfocado en el desarrollo de herramientas avanzadas para Minecraft, con énfasis en rendimiento, estabilidad y optimización de mecánicas. Se centra en crear soluciones técnicas para entornos anarchy y usuarios que buscan mayor control sobre el juego.
