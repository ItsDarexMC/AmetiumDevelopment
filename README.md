# developments

repositorio central para la gestión y despliegue de soluciones de software de **itsdarexmc**. este espacio sirve como el hub principal para arquitecturas modulares y extensiones de cliente dentro del ecosistema de minecraft utility.

---

## proyectos

### amethyst development
addon de alto rendimiento para **meteor client 1.21.11**. especializado en la automatización de módulos de combate y optimización de sistemas para servidores de anarquía.
* **estado:** estable.
* **arquitectura:** java / fabric loader.

### helium development
nueva generación de sistemas utilitarios. enfocado en la ligereza del código y una experiencia de usuario simplificada.
* **estado:** desarrollo activo.
* **arquitectura:** java / fabric loader.

---

## estructura del repositorio

la disposición de archivos está optimizada para el despliegue automático mediante **github pages**:

* `index.html`: hub de selección principal (interfaz de usuario).
* `amethyst.html`: landing page técnica para el proyecto amethyst.
* `helium.html`: landing page técnica para el proyecto helium.
* `/amethystclient`: directorio para procesos de adquisición de licencia amethyst.
* `/heliumclient`: directorio para procesos de adquisición de licencia helium.

---

## especificaciones técnicas

la interfaz web utiliza un stack moderno enfocado en la legibilidad y el rendimiento:
* **layout:** responsive css con filtrado backdrop (glassmorphism).
* **fuente:** poppins (300, 500, 700).
* **despliegue:** entornos integrados de github actions.
