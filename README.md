## Ametium Development

Ametium Development es el proyecto principal enfocado en el desarrollo de herramientas avanzadas para Minecraft, Se centra en crear soluciones técnicas para entornos anarchy y usuarios que buscan mayor control sobre el juego.

## Ametium Addon

Ametium Addon es un addon para Meteor Client, enfocado en anarquía, utilidades y multiples modos de juego. Incluye funciones optimizadas para mejorar la experiencia en estos modos.

## Ametium Client

Cliente para Minecraft 1.21.1 enfocado en entornos anarchy, mejoras de combate, movimiento modificado y utilidades avanzadas que amplían el control y rendimiento del jugador. Sin fecha de lanzamiento estimada.
